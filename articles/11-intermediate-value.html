Intermediate value :indeterminate

The :indeterminate pseudo-class matches UI elements whose value is in an indeterminate state. An example of an element that can be in an indeterminate state is radio and checkbox elements. They can be checked, unchecked or neither (indeterminate).

Browser Support

Chrome Firefox Safari Opera IE Edge
Desktop yes yes yes yes yes yes
Mobile yes yes yes yes yes yes

Syntax

          
            :indeterminate {
              /* Style properties */
            }

Example

          
            /* if the radio or checkbox is in an indeterminate state, the label will have a red background. */

            :indeterminate + label {
              background-color: red;
            }
