Language :lang(*-DE)

The :lang() pseudo-class was introduces in CSS2. Level 4 selectors adds wildcard matching. The language pseudo-class represents an element that is in one of the languages listed as an argument.

Browser Support

Chrome Firefox Safari Opera IE Edge
Desktop no no no no no no
Mobile no no no no no no

Syntax

          
            E:lang(*-DE) {
              /* Where this matches anything ending in "*-DE". */
            }

Example

          
            /* This matches all p elements with a lang attribute ending in -DE */

            p:lang(*-DE) {
              background-color: green;
            }
