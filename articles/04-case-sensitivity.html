Case-Sensitivity E[foo="bar" i]

The case-sensitivity selector matches any element whose specified attribute is equal to a value of any case combination.

Browser Support

Chrome Firefox Safari Opera IE Edge
Desktop yes yes yes yes no no
Mobile no yes yes yes no no

Syntax

          
            E[attr="val" i]
              /* Where "i" stands for case-insensitive. */
            }

Example

          
            /* all input elements containing a value of name, Name, NAME, etc. */

            input[value="name" i] {
              border: 4px solid #00aa00;
            }
